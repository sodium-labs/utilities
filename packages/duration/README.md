## About

Time and duration utility package.

## Installation

Node.js 18 or newer is required.

```sh
npm install @sodiumlabs/duration
```

## Usage

```ts
import { Time } from "@sodiumlabs/duration";

setInterval(() => {
    console.log("Called every 2 hours!");
}, Time.Hour * 2);
```
